Shoppy is an e-commerce iOS application that allows users to shop online, browse product catalogs, create wish lists, add items to a cart, and complete purchases.


## Features:
- Authentication With [Email & Password] or [Google account].
- Home page that contains sections of different categories.
- Searching for any product.
- A Wish list and general lists that you can create.
- Cart, checkout, and address views to complete your purchases
- A profile view contains your orders and lists.


## Technologies: 
- UI Framework: UIKit
- Architecture pattern: MVVM (Model-View-ViewModel)
- Authentication: Firebase Authentication
- Cloud database: Firestore
- Reactive Programming library: RxSwift
- Selecting Address Location: MapKit
- Async Images downloader: SDWebImage
- Asynchronous Programming: Swift Concurrency
- Products API: Shopify
